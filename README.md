# 🏏 one4sports - Premium Sports Shop

ही माझी पहिली ई-कॉमर्स वेबसाईट आहे. यामध्ये क्रिकेट आणि फूटवेअरची माहिती आणि ऑर्डर्सची सोय आहे.

## 🚀 फीचर्स (Features)
- **Premium UI:** Myntra सारखा लुक.
- **Exact Search:** Amazon सारखी सर्च सिस्टिम.
- **Live Orders:** ऑर्डर्सची नोंद MongoDB मध्ये होते.

## 🛠️ टेक्नॉलॉजी (Tech Stack)
- Node.js & Express
- MongoDB Atlas
- HTML, CSS & JS
